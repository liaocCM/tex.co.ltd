# User Flows

## Create TODO
1. User clicks Add.
2. Modal opens.
3. User enters title/notes and saves.
4. TODO appears in list.

## Update TODO
1. User clicks Edit.
2. Modal opens with values.
3. User saves changes.
4. List updates and persists.

## Complete TODO
1. User toggles checkbox.
2. Item updates to completed state.

## Delete TODO
1. User clicks Delete.
2. Item removed from list.

## Filter TODOs
1. User selects All/Active/Completed.
2. List updates accordingly.
