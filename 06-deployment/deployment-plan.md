# Deployment Plan

## Environments
- Local development only for MVP.

## Prerequisites
- Go 1.21+
- Node.js 18+
- SQLite file access to `data/`

## Steps
1. Start backend: `go run .` in `04-implementation/backend/`.
2. Start frontend: `npm install && npm run dev` in `04-implementation/frontend/`.
3. Access UI at the Vite dev server URL.
